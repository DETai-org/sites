# /det/concept

Раздел `/det/concept` — это введение и ядро концепции DET. Он объясняет основу
подхода и подводит к трем постулатам, которые раскрывают ключевые идеи.

## Роль постулатов

Постулаты — это pillar pages, которые читаются как самостоятельные страницы и
вместе образуют каркас смыслов. **Pillar page = глава книги в сжатом виде.**

Связанные шаблоны и компоненты описаны в
[`components/concept/README.md`](../../../../components/concept/README.md).
