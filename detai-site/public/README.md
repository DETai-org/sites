# Публичная директория с файлами, доступными напрямую по URL.  


Используется для:

- favicon, иконок, логотипов  
- robots.txt и sitemap.xml  
- изображений, фонов и медиа  
- PDF-файлов, документов, статических данных  
- любых ассетов, которые не должны компилироваться Webpack'ом

Файлы в `public/` доступны по прямой ссылке:
пример: `/images/products/bot1.png`
