# components/concept

Здесь находятся общие компоненты и шаблоны для страниц постулатов (pillar pages)
раздела `/det/concept`.

## Что важно знать

- Страницы постулатов живут в `app/(det)/det/concept/*`.
- Тексты постулатов хранятся прямо в файлах `page.tsx` конкретных страниц, без
  динамических `[slug]` и скрытых data-слоёв.

См. также описание раздела концепции в
[`app/(det)/det/concept/README.md`](../../app/(det)/det/concept/README.md).
