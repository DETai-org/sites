# ## Кнопки Button.tsx

### **Функции кнопок (3 состояния)**

Каждая кнопка имеет три смысловые функции поведения:

* **Hover** — реакция при наведении.
* **Active** — реакция при нажатии.
* **Focus** — состояние при выделении клавиатурой.

### **Типы кнопок (5 уровней действий)**

Кнопки делятся на пять уровней по важности и назначению:

1. **Primary** — главное действие на экране.
2. **Secondary** — второстепенное действие рядом с основным.
3. **Tertiary** — текстовая, минимальная кнопка.
4. **Ghost / Premium** — особая премиальная кнопка с эффектами (искры, свечение).
5. **Destructive** — действия, требующие осторожности (удалить, отключить).
