# Взаимопомощь как фактор эволюции

Для Маркса, человек – первично социальное существо. Он нуждаеться в других людях не потому что они средства для удовлетворения его желаний, а просто потому, что человек есть человек, и он только тогда обладает человеческой завершоностью, когда связан с другими людьми. Гёте утверждал "человек знает себя настолько, насколько знает мир. Он знает мир только внутри себя и осознаёт себя только внутри мира."
Эрих Фромм говорил так: "Полностью развитой и, стало быть, здоровый человек – это человек продуктивный, человек, подлинно заинтересованный в мире, откликающейся на его запросы" (Э. Фромм, Из плена илюзий с. 85-88)
