# Централизованные данные публикаций

Эта директория собирает единую схему и данные научных материалов DET. Все страницы и списки публикаций читают информацию только отсюда, поэтому для добавления новой статьи достаточно:

1. положить PDF в `public/documents/research/`;
2. добавить объект в [`publications.data.ts`](./publications.data.ts) согласно типам из [`types.ts`](./types.ts).

## Файлы
- **types.ts** — объявляет типы `Publication` и `PublicationType` с обязательными полями (slug, авторы, год, аннотация, ссылки).
- **publications.data.ts** — хранит массив `publications` с данными обо всех материалах.
- **publications.utils.ts** — утилиты для выборки публикаций (по типу, по slug, весь список) для страниц и компонентов.

## Как это работает
- Страницы `/det/publications` и `/det/publications/[slug]`, а также блок «Публикации» на дет-странице используют функции из `publications.utils.ts`.
- Заголовок публикации всегда ведёт на страницу `/det/publications/[slug]`, откуда уже доступна ссылка «Скачать PDF» и внешние ссылки (например, DOI).
- Добавление нового элемента в `publications.data.ts` автоматически обновляет общий список и вкладки без правок компонентов.
