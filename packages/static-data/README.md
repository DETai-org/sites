# Static Data

Этот каталог содержит статические данные и политики, используемые в монорепозитории `sites`.

## Структура

- `authors.json` — универсальный реестр авторов для постов и научных публикаций: здесь хранятся имена и фамилии участников команды, публикующих контент.
- `post_documents/` — источник истины для таксономии: per-type JSON файлы (например, `categories.json`, `rubrics.json`, `keywords.json`), где каждая сущность хранится в одном объекте вместе со всеми переводами и локализованными `routeSlugs` (см. `policies/route_slugs_policy.md`).
- `policies/` — технические политики и правила работы с данными (включая политику локализованных slug постов и политику хранения обложек постов).
- `research_publication/` — данные и документы, связанные с публикациями.
