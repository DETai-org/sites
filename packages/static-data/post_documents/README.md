## Типы документов
На сайте существуют основные типы: которые при этом размещены в разных местах по крайней мере посты - Это всё что в блоге и туда же ещё и цитаты добавляются то есть тип документа цитата
А вот research_publication это научные публикации у них будут свои поля которые нужно будет переводить на разные языки и эти документы будут храниться в разделах сайта которые связаны с публикациями например detai-site/public/documents/research/ но контракты по этим по типам в названий внутри этих этого типа документа будет позже пока только есть готовые сущности для типа документа пост
post — основной тип публикаций (посты, эссе, заметки);
research_publication — научные публикации (статьи, диссертации, тезисы);
quote — цитаты.
