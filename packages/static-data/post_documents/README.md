## Типы документов
Сейчас на сайте выделяются основные типы документов. Блоговые материалы оформляются как `post` — это все публикации в блоге (посты, эссе, заметки), а также цитаты, которые вынесены в отдельный тип `quote`. Научные материалы относятся к типу `research_publication`: для них предусмотрены собственные поля, которые будут переводиться на разные языки. Такие документы будут храниться в разделах сайта, связанных с публикациями, например `detai-site/public/documents/research/`. Контракты и точные требования к структуре `research_publication` появятся позже; на данный момент полностью проработаны сущности только для типа `post`.

- `post` — основной тип публикаций (посты, эссе, заметки);
- `research_publication` — научные публикации (статьи, диссертации, тезисы);
- `quote` — цитаты.

## Локализация таксономии
Файлы таксономии в этой директории (rubrics, categories, keywords, cycles) используют локализованные поля в формате `{ru/en/de/fi/cn}` для `label`, `description` и связанных текстовых полей, чтобы все языки были заполнены в одном объекте. Это облегчает единый контроль переводов и синхронизацию данных между сайтами.

## Загрузка данных
Для чтения и нормализации таксономии используйте `index.ts` в этой директории — он предоставляет единый загрузчик и типы, гарантируя наличие 5 языков.
