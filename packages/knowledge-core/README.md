# knowledge-core

Базовый пакет для описания схем, политик и словарей публикаций.

## Структура

- `schemas/` — схемы данных.
- `controlled_vocabularies/` — контролируемые словари.
  - `publications/` — словари для публикаций.
    - `rubrics.json` теперь содержит отдельные блоки рубрик для профессионального и персонального сайта.
    - `cycle.json` — тематические циклы персонального сайта, оформленные как отдельный словарь.
    - `telegram-notes.md` — вынесенные тексты и заголовки постов из тематических рубрик WordPress-архива.
- `policies/` — политики и правила.
- `mappings/` — сопоставления с внешними схемами.
- `routing/` — правила маршрутизации публикаций.

## Статус

Пока используется как каркас с заглушками для последующего наполнения.
